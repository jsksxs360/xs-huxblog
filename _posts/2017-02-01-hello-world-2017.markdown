## 基本 Markdown 语法

##  各种语言的代码渲染

本博客使用 [highlight.js](https://highlightjs.org/) 对代码进行进行渲染，选用 Github 主题，下面为展示效果：

**C**

```c
#include <stdio.h>
int main()
{
    printf("Hello World!/n");
    return 0;
}
```

**Java**

```java
class Hello
{
    public static void main(String ars[])
    {
        System.out.print("Hello World!/n");
    }
}
```
